# 🏥 Medical AI Chatbot

An intelligent Medical AI Chatbot built with **Groq LLM** and **RAG (Retrieval-Augmented Generation)** technology.

## ✨ Features
- 💬 Beautiful chat UI (works directly in browser)
- 🧠 RAG-powered answers from medical knowledge base
- ⚡ Super fast responses powered by Groq
- 🚨 Emergency detection and alerts
- 📱 Mobile friendly
- 🔒 No data stored — fully private

## 🛠️ Tech Stack
- **LLM**: Groq (llama-3.1-8b-instant) — Free
- **RAG**: FAISS + Sentence Transformers
- **UI**: Pure HTML, CSS, JavaScript
- **Backend**: Python

## 🚀 How to Run

### Option 1 — Browser UI (Easiest)
1. Open `medical_ui.html` in your browser
2. Enter your free Groq API key from https://console.groq.com
3. Start chatting!

### Option 2 — Terminal
```bash
pip install groq sentence-transformers faiss-cpu
python medical_chatbot_groq.py
```

## 🔑 Get Free API Key
1. Go to https://console.groq.com
2. Sign up free
3. Click API Keys → Create API Key
4. Copy and paste into the chatbot
